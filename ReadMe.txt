Python Project to send GET requests over HTTP protocol to Web Connected Devices as well as custom server program using sockets.

Start Server with "python HTTPServer.py <ipaddress>:<portnumber>"

Start client with "python HTTPClient.py [PUT] URL [FILEPATH]
    Only use PUT when conencting to server program
    URL is for URL to file on server you are connecting to or..
    URL is for the URL where you want to store your file on your server when using PUT
    FILEPATH is the filepath to the file you want to send using put
